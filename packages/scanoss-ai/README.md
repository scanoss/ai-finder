# scanoss-ai

CLI for SCANOSS AI artifact scanner.

## Installation

```bash
pip install scanoss-ai
```

## Usage

```bash
scanoss-ai scan /path/to/project
scanoss-ai identify model.gguf
scanoss-ai kb init
```

## Global Options

| Option | Description |
|--------|-------------|
| `--version` | Show version and exit |
| `--no-telemetry` | Disable anonymous usage telemetry for this session |
| `--help` | Show help message |

## Telemetry

This tool collects anonymous usage telemetry. See [TELEMETRY.md](../../docs/TELEMETRY.md) for details and opt-out instructions.
